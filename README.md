# Android 超级实用日志调式工具-HlLog
🔥 Hllog 超级实用的android 日志打印工具，支持文件打印，app上视图日志打印，控制台日志打印；Hllog具有高可扩展性，可以自己实现自己的打印器。
### 特点功能：
- 支持在手机app界面上直接实时查看日志；
- 支持控制台查看日志；
- 支持打印文件日志；
- 支持打印堆栈调用信息及深度；
- 支持配置打印线程信息；
- 高可扩展性，可定义自己的打印器；
